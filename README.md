# 📝 To-Do List Web App

A simple and clean to-do list web application built using HTML, CSS, and JavaScript.
It allows users to manage daily tasks with an interactive and minimal interface.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Clean and modern UI design
* Responsive layout

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript


## ▶️ How to Run

1. Download or clone the repository
2. Open `to-do.html` in your browser


## 🎮 Usage

* Enter a task and click **Add**
* Click on a task to mark it as completed
* Click **Done** to remove the task


## 🎯 Future Improvements

* Save tasks using localStorage
* Add edit functionality
* Improve animations and UI

---

This project was built to practice DOM manipulation and basic frontend development.
